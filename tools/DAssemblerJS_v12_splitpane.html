Apple II 6502 DASM v12

Input formats: legacy 5-byte CPU trace records (adrLo adrHi opcode op1 op2, base64 encoded), enhanced 10-byte CPU trace records (adrLo adrHi opcode op1 op2 A X Y P SP, base64 encoded), or a regular memory dump such as F800: 4A 08 20 47 F8. DASM v12 keeps two independent input panes and two independent disassembly panes, so two boot logs or memory dumps can be viewed side-by-side.
