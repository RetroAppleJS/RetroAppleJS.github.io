JSON Base64 Image Extractor / Replacer

Standalone local browser tool. It scans a large JSON file without JSON.parse() and looks for objects containing "uuid" followed by "url": "data:image/...;base64,...".

1. Extract embedded images from JSON

Optional helper mode. Images are saved as <uuid>.<extension>.

No folder selected. If unsupported, browser downloads will be used.

2. Rebuild JSON with edited/compressed images

Select the original JSON and the edited image files. Each edited image must keep the UUID as filename stem, for example c22f5acf-8e3a-40ce-a194-c16f7a0dd275.png, .jpg, or .webp.

In Chrome/Edge this can select a folder. In other browsers, use a normal multi-file selection if available.
