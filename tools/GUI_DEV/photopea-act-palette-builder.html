Photopea ACT Palette Builder

Paste RGB colors, preview them, and download a Photopea / Photoshop .act color table.

Accepted per line: rgb(r,g,b), rgba(r,g,b,a), #RRGGBB, #RRGGBBAA, #RGB, r,g,b, r,g,b,a, or r g b. Alpha values are accepted but ignored by ACT export. Blank lines and // comments are ignored.

This can read a 768- or 772-byte ACT file and put the used colors back into the textarea.
